General Questions

ASO-Suite Pro is a comprehensive WordPress plugin that provides 22+ modules for admin security, optimization, and control. It includes a free edition with 9 core modules and a Pro edition with all features unlocked.

Yes, the basic edition is completely free and includes 9 essential modules. The Pro edition costs $49 and includes all 22+ modules plus premium features. Both include free updates forever.

ASO-Suite Pro is perfect for WordPress site owners, developers, agencies, and anyone who wants professional-grade security, optimization, and admin control over their WordPress sites.

The free edition is available on WordPress.org under the GPL v2 license. The Pro edition is proprietary but includes all free edition features plus premium modules.

Installation & Compatibility

ASO-Suite Pro requires WordPress 6.0 or higher and PHP 7.4 or higher. We test extensively with PHP 8.3.23 for maximum compatibility.

Simply go to Plugins ‚Üí Add New in your WordPress admin, search for "ASO-Suite Pro," and click Install. Then activate it. For manual installation, upload the plugin folder to wp-content/plugins/ and activate in WordPress.

Yes, ASO-Suite Pro is compatible with all modern WordPress themes. It's a backend plugin that only modifies WordPress admin functionality, not your frontend theme.

No, ASO-Suite Pro is optimized for performance. It only loads in the WordPress admin area and has minimal impact on frontend performance. In fact, image optimization and other features can improve site speed.

Features & Functionality

The Free edition includes 9 core modules with basic security and admin tools. The Pro edition unlocks all 22+ modules including advanced security (CAPTCHA), automation (ChatPulse, bulk creator), and premium features like Google Sheets integration.

Yes. Both editions include basic export functionality. The Pro edition includes a complete backup and export system with advanced options and role-based access control.

CAPTCHA login protection is a premium feature available only in the Pro edition. It helps protect against brute force attacks on your WordPress login page.

Google Sheets integration (SheetableDataTable) is available in the Pro edition and allows you to create dynamic tables on your site directly from Google Sheets data.
